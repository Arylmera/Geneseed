# {{SKILL}}: crash-course

> {{DESC_CRASH_COURSE}}

**Trigger:** the user wants to go from zero to *functional* in a skill fast — "teach me X", "get me up to speed on X", "I have limited time to learn X". For practising a concept until it's reflex use [drill](drill.md); for making confusing material click use [decode](decode.md); for gathering facts from the web use [research](research.md).

## Procedure
1. Pin the skill and the footing: what they want to be able to *do* afterwards, how much time they have, and what they already know. If unclear, ask once, then proceed.
2. Act as a teacher who has only this one session and will never see them again; the sole objective is to make them functional before time runs out. Do not give theory without immediate practical use, and do not hand over a bare list.
3. Name three things explicitly: **what to learn first**, **what to ignore completely** (for now), and **the single exercise that, done once, puts them ahead of most people who have studied this for months**.
4. Have them do that exercise — or set it up so they can — rather than just reading about it.
5. Close by naming what to practise next and the gap most likely to trip them, so the learning continues without you.

## Done when
- The user knows what to learn first, what to skip, and has the one high-leverage exercise in hand — functional, not just informed.

## Self-improvement

Close each run with one beat of reflection on the {{SKILL}} itself:
- A step misled, a needed step was missing, or the trigger fired wrongly — that
  is a flaw in this file. Propose the exact edit (trigger, procedure, or
  done-when) and apply it with the user's assent ({{LAW}} II).
- A lesson that is *not* a flaw in this file goes to {{MEMORY}} only if it
  clears {{LAW}} VI's bar: it would change how a future session behaves, and a
  fresh read of the repo would not re-derive it. Update an existing memory over
  adding one; when in doubt, leave it out.
- No friction, nothing learned — move on; this loop earns no ceremony. Most
  runs end here.
