# {{SKILL}}: decode

> {{DESC_DECODE}}

**Trigger:** the user hands over content that confuses them and wants it to *click* — "this is confusing, help me understand it", "break this down for me", "explain this passage". For a from-zero overview of a whole skill use [crash-course](crash-course.md); for practising a concept until it is reflex use [drill](drill.md); to convert a PDF or web doc into readable form first use [ingest](ingest.md).

## Procedure
1. Read the content the user is stuck on. Before explaining anything, find the one sentence that, once understood, makes the rest fall into place.
2. Explain only that sentence first — using an everyday analogy, with no technical terms.
3. Confirm that landed, then let the rest of the content follow from it rather than re-explaining every line.
4. Ask three questions that only someone who truly understood it could answer. Do not continue until they pass all three.
5. If they miss one, return to the analogy and re-anchor before asking again — don't just give the answer.

## Done when
- The user has the keystone idea in plain terms and has answered all three comprehension questions correctly.
