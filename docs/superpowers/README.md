# Superpowers — exploratory archive

These are **early implementation plans and a design spec** for the Geneseed web
console (and the "Cultivar" re-skin), written in a checkbox-driven planning format
during a multi-agent design pass.

**The work they describe has shipped.** The web console — Dashboard, Library, Graph,
Docs, Specs, Changes, Doctor, Themes, Settings — is live under [`web/`](../../web/), and
its design record now lives in the dated specs at [`../specs/`](../specs/). These files
are kept as **history**, not as live specifications:

- [`plans/2026-06-12-web-ui.md`](plans/2026-06-12-web-ui.md) — the web UI build plan
- [`plans/2026-06-12-cultivar-reskin.md`](plans/2026-06-12-cultivar-reskin.md) — the "Cultivar" console re-skin plan
- [`specs/2026-06-12-web-ui-design.md`](specs/2026-06-12-web-ui-design.md) — the web UI design spec

For the current state of the harness and where each feature's rationale lives, see
[`SHIPPED.md`](../../SHIPPED.md). For the canonical design contract, see
[`DESIGN.md`](../../DESIGN.md).
