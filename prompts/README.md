# Install prompts

Theme-flavoured one-shot prompts pasted into an agent to install Geneseed.

Only **imperial** and **neutral** are maintained — they are the two reference
themes (the maximal and the plain rendering). The other six themes install the
same way: use `install.neutral.md` and pass the theme name where it says
`neutral` (e.g. `./geneseed setup` or `./upgrade.sh main cyberpunk`). This gap
is intentional, not an oversight — six near-identical copies would drift.
